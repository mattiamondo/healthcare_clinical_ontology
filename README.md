# Healthcare Clinical Ontology

OWL 2 DL healthcare-clinical ontology (patients, diseases, symptoms,
treatments, medical staff, facilities and care encounters). Project for the
**Knowledge Representation** exam — MSc in Artificial Intelligence,
University of Verona, A.Y. 2025–2026.

- **Format:** OWL 2 DL
- **Editor:** Protégé
- **Reasoner:** HermiT (consistent, no unsatisfiable classes)
- **File:** `healthcare_VR538106.rdf`
